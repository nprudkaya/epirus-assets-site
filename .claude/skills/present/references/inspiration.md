# Inspiration Catalog

Reference this when planning interactive elements. These are **seeds, not boundaries** — combine, remix, and invent beyond this list.

## Opening Patterns

| Pattern | Description | Impact |
|---------|-------------|--------|
| Terminal Boot | Monospace text appearing line-by-line, simulating a system starting up | Sets technical tone, builds anticipation |
| Title Detonation | Title word starts tiny/blurred, explosively scales to full size | Immediate energy, impossible to ignore |
| Constellation Build | Points of light connect to form the title or logo | Elegant, memorable, suggests interconnection |
| Typewriter Reveal | Characters appear one by one with cursor | Classic, personal, intimate |
| Glitch Intro | Text distorts, flickers, then resolves into clarity | Edgy, modern, suggests digital transformation |

## Interactive Slide Types

| Type | Mechanic | Best For |
|------|----------|----------|
| Flip Cards | Click to rotate 180°, reveals back content | Comparing before/after, Q&A reveals, benefit details |
| Explorable Diagram | Clickable nodes in a system diagram, each reveals detail panel | Architecture, workflows, org structures |
| Live Terminal | Simulated code typing + output appearing | Technical demos, showing automation |
| Chat Replay | Messages appear in sequence like a real conversation | Showing AI interactions, user stories |
| Drag Comparison | Slider divides two states, drag to compare | Before/after, two approaches side-by-side |
| Counting Dashboard | Numbers count up from 0 to target on slide entry | Metrics, impact numbers, statistics |
| Timeline Scrubber | Horizontal timeline, click points to jump to moments | History, project evolution, chronology |
| Card Deck | Stack of cards, click to fan out and explore each | Team members, features, portfolio items |
| Hover Anatomy | Regions of an image/diagram reveal labels on hover | Product breakdowns, system components |
| Progressive Build | Each click/keypress adds one more element to the slide | Building complexity, layered explanations |

## Transition Ideas

| Transition | Description |
|-----------|-------------|
| Dissolve Particles | Current slide breaks into particles, next slide assembles from them |
| Depth Zoom | Camera zooms into a detail of current slide, which becomes the next slide |
| Page Turn | 3D page-flip effect |
| Morph | Shared elements smoothly transform position/size between slides |
| Blackout Beat | Brief black screen (0.3s) for dramatic pause between sections |
| Directional Wipe | Content sweeps away in the direction of navigation |

## Background Effects

| Effect | Implementation | Weight |
|--------|---------------|--------|
| Star Field | Canvas with drifting dots of varying opacity | Light |
| Particle Drift | Floating particles that respond to mouse position | Medium |
| Gradient Pulse | Subtle color shifts in the background over time | Light |
| Grid Matrix | Faint grid lines that pulse or shift | Light |
| Aurora | Slowly morphing color blobs (use sparingly, can feel generic) | Medium |
| Noise Texture | Subtle film grain overlay for analog warmth | Light |

## Emotional Pacing

Structure the presentation's energy like a story:

```
Energy
  ▲
  │    ╱╲        ╱╲    ╱╲
  │   ╱  ╲  ╱╲ ╱  ╲  ╱  ╲  ╱╲
  │  ╱    ╲╱  ╲    ╲╱    ╲╱  ╲
  │ ╱                          ╲
  └──────────────────────────────►
  Open  Build  Peak  Breathe  Finale
```

- **Open**: Hook them (high energy or intriguing calm)
- **Build**: Layer content, increase complexity
- **Peak**: The "wow" moment — your best interactive slide
- **Breathe**: Reflective slide, quote, or pause
- **Finale**: Memorable close, callback to opening, clear takeaway
